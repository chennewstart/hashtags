Built with the POWER of BOWER!

Bower is a package manager for the web. Bower depends on node and npm. It can be installed globally using npm: 

npm install -g bower

Download all your favorite frontend applications using a simple command interface. For example: 

bower install bootstrap

Bootstrap your app with Twitter Bootstrap.

Bootstrap is the most popular frontend framework for developing responsive, mobile first projects on the web. Easily get started building beautiful, interactive web pages no matter your skill level.

Create powerful HTML presentations with Reveal.JS.

Reveal.js is a framework for easily creating beautiful presentations using HTML. There is even a fully-featured visual editor. You can try it out at http://slid.es.

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. New to Git? Try it!
